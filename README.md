# OpenGl-project

* WEATHER switching: with key "P"
* LIGHTS ON and OFF - "L"
* LIGHTS BRIGHTNESS LEVELS - "+" AND "-"
* MOVING AROUND WITH W,A,S,D keys


USED libriaries:
ASSIMP, 
SDL2,
GLEW-2.2.0,
GLM

inside Linker -> Input -> Additional Dependecies need to add : opengl32.lib;glu32.lib

house, ko, light, zahaLight need to be put into a folder called "models".
